About the Project

This website is part of a mini project in data analysis, developed by Adithyan, Akshy, and Febin. The main purpose of this project is to provide users with a camera recommendation system based on data analysis techniques.

The project was completed as part of the academic curriculum in the field of data analysis. The aim is to help users explore different camera models and find similar products based on their preferences.

Key Features of the Website

  • Camera Recommendation System using Cosine Similarity
  • User-friendly interface with dark mode for better user experience
  • Dynamic display of camera features such as resolution, ISO range, video resolution, and more
  • Responsive design for all device types
  • Search functionality for finding specific camera models
  • Information about each camera's technical specifications and price

We hope you find this website helpful in your search for the perfect camera!
